## ゲーム概要

1 つのボードを 2 人のプレイヤーが共有して対戦するオリジナルルールのテトリスです。効率的なライン消し方法を見極め、より多くのラインを消したプレイヤーの勝ちとなります。

※1 プレイヤーでのゲームプレイも対応しています。

## 操作方法

### 1P

-   右移動: D キー
-   左移動: A キー
-   下移動: S キー
-   左回転: Q キー
-   右回転: E キー

### 2P

-   右移動: 右カーソルキー
-   左移動: 左カーソルキー
-   下移動: 下カーソルキー
-   左回転: 右 ALT キー
-   右回転: 右 CTRL キー
